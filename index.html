Sobre este proyecto

Objetivos de aprendizaje

Con este proyecto buscamos desarrollar competencias digitales básicas que nos permitan:

  • Comprender la estructura básica de una página web
  • Aplicar etiquetas HTML fundamentales
  • Utilizar CSS para dar estilo a nuestro contenido
  • Organizar información de manera digital
  • Presentar contenido de forma atractiva y funcional

Reflexión sobre Cultura Digital

Este trabajo nos ha permitido entender mejor cómo funcionan las páginas web que usamos diariamente. En la era digital, comprender estos conceptos básicos es fundamental para ser usuarios críticos y conscientes de la tecnología.

Aprendizajes clave

Durante el desarrollo de este proyecto descubrimos que crear una página web requiere:

  1. Planificación previa del contenido
  2. Organización lógica de la información
  3. Paciencia para resolver errores
  4. Creatividad para el diseño
  5. Trabajo en equipo
